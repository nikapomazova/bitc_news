**Bitcoin & News Sentiment Dashboard**

Interactive app that analyzes the relationship between Bitcoin price movements and news sentiment over time.
The application fetches Bitcoin price data and Bitcoin-related news articles, computes daily average news negativity, and visualizes both series on a single chart.

What is inside this project?
- Real-world time-series data
- Data sampling using pandas
- Sentiment analysis on news data
- Interactive data visualization with Plotly
